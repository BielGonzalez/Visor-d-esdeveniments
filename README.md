### Quin event succeeix quan s'inicia Windows? 6005 (event log service started)

### Quin event succeeix quan es crea un compte d'usuari? 4720 (a user account was created).
### Quin event succeeix quan un usuari falla un login? 4625 (failed login attempt).
### Quin event succeeix quan un usuari canvia l'hora del SO?  4616 (time was changed).
